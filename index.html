Phoros

Sources for the study of Athenian tribute

View My GitHub Profile

The Phoros project is creating new editions of the epigraphic sources for the study of tribute in ancient Athens, based on new, openly licensed photography. All of our editorial work is freely available fro github as simple text files or TEI-conformant XML editions.
